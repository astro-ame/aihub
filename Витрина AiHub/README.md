# Ai Hub Market — бот магазина нейросетей

После `/start` показывается главное меню: изображение, приветственный текст и кнопки.

## Данные (сохранены в проекте)

- **Токен бота:** в `config.py` / `.env`
- **ID менеджера поддержки:** `7210745918`
- **ID канала:** `3821110723`, ссылка для кнопки «Новости Ai Hub» — `https://t.me/c/3821110723`
- **Pally API:** токен и база в `config.py`, гайд — https://pally.info/merchant/api
- **Изображение приветствия:** https://disk.yandex.ru/i/mnaLpzGaoAA52Q

## Запуск

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python bot.py
```

Опционально: скопируйте `.env.example` в `.env` и измените значения (токены, ссылки).

## Кнопки главного меню

| Кнопка | Действие |
|--------|----------|
| Профиль | Заглушка (позже — покупки и доступы) |
| Автовыдача | Заглушка (пояснение про автовыдачу) |
| Все нейросети | Заглушка (позже — каталог и оплата через Pally) |
| Новости Ai Hub | Сообщение с кнопкой «Открыть канал» → открывается ваш ТГ-канал |
| Поддержка | Кнопка «Написать в поддержку» → чат с менеджером |
| О боте | Текст про бот и принцип работы |

Дальше можно добавить каталог товаров, интеграцию с Pally и автовыдачу.
